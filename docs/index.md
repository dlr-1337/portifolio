## Project Overview

This repository contains a single-page developer portfolio for Otavio Queiroz. The app is built with Next.js App Router and renders one polished homepage from typed content stored in source control. There is no committed backend, database, or internal API layer; the current system is primarily a static, SEO-aware presentation surface.

## Quick Reference

- **Tech Stack:** Next.js 16, React 19, TypeScript, Tailwind CSS 4, `motion/react`, Vitest, Testing Library
- **Entry Point:** `src/app/page.tsx`
- **Architecture Pattern:** server route composition + single client presentation component + centralized content module
- **Content Source:** `src/content/site.ts`
- **Main UI Component:** `src/components/portfolio/portfolio-page.tsx`
- **Deployment Status:** no committed deployment pipeline detected

## Getting Started

### Prerequisites

- Node.js current LTS compatible with Next.js 16
- npm
- dependency installation via `npm install`

### Setup

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Validate

```bash
npm run lint
npm run test
npm run build
```
